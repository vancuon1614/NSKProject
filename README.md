## ỨNG DỤNG XEM BẢN BỒ

**Bản đồ Việt Nam sau khi sát nhập**

## Ứng dụng có hỗ trợ các chức năng:
- ZoomIn, ZoomOut
- Ruler (đo khoảng cách)
- Rediction (tìm đường đi)
- Polygon (Vẽ đo diện tích)
  
**Ứng dụng sử dụng ngôn ngữ React Native kết hợp thư viện MapLibre**

## Để sử dụng ứng dụng, Cần:
1. Init project: npx @react-native-community/cli@latest init name_project
2. Clone Repo về máy qua đường dẫn ở link Repo
3. Cần Install package trước thông qua Native Package Manager (npm) với lệnh: npm install @maplibre/maplibre-react-native
4. Chạy thử ứng dụng lần dầu với lệnh: npm run android hoặc npx react-native run-anđroid
